# Netflix Data Analysis Dashboard (Power BI Project)

## Project Overview  
This project focuses on analyzing Netflix movies and TV shows data to uncover trends, audience preferences, and content performance using Power BI. The goal is to derive meaningful insights from entertainment data to support decision-making.

---

## Business Problem  

With the rapid growth of streaming platforms, understanding content trends and audience behavior is crucial.

### Key Question  
How have content characteristics, ratings, and audience engagement evolved over time on Netflix?

## Key Questions Addressed  

- Which is the best movie or show overall based on ratings and popularity?  
- How many movies were released across different years?  
- How have IMDb scores changed over time?  
- Have IMDb voter counts increased over time?  
- How has runtime changed across decades?  
- Does age certification affect ratings?  

## Tools & Techniques  

- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Data Visualization  

## Key Insights  

- Content release has significantly increased over the years  
- IMDb ratings have remained relatively stable over time  
- Viewer engagement (IMDb voters) has increased, indicating growing audience interest  
- Runtime patterns have evolved across decades  
- Age certification influences content ratings and audience preferences  

## Outcomes  

- Identified trends in content production and audience engagement  
- Provided insights into how ratings and runtime have evolved  
- Helped understand factors affecting content performance  
- Enabled data-driven storytelling through interactive dashboards
